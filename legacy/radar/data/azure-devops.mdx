Azure DevOps is a comprehensive suite of integrated development and project management tools including ticket boards,
repositories, and deployment pipelines. However, many of the tools are lacking features in their competitive
counterparts, such as pull-request closed triggers.

*June 28, 2022:* Principle is continuing to use Azure DevOps for closed-source projects, but is preferring GitHub
Actions due to the variety of additional triggers.
