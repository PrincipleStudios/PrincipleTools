OpenAPI (aka Swagger) is a powerful tool for sharing contracts of HTTP APIs between teams. 

*June 28, 2022:* Principle Studios uses OpenAPI to generate both server-side and client-side code along with
strongly-typed languages to ensure both front-end and back-end teams remain in sync.
